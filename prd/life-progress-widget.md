# 需求文档：人生进度小组件 (Life Progress Widget)

## 1. 背景与目标
根据用户输入的出生日期，计算并展示用户在假设的30000天寿命中，已经度过的天数和剩余的天数。小组件需要美观，支持Egern官方小组件配置语法。

## 2. 功能设计
- **输入参数（环境变量）**:
  - `BIRTH_DATE`: 出生日期，只需 YYYY-MM-DD 格式（如 "1990-01-01"）。
  - `LIFE_EXPECTANCY`: 预期寿命天数，默认为 30000。
  - `TITLE`: 组件标题，默认为 "人生进度"。
  - `COLOR_1` & `COLOR_2`: 背景渐变色，默认提供一套美观的渐变色组合。
- **UI 布局**:
  - **背景**: 动态渐变背景。
  - **顶部**: 展示图标 + 标题（如 "⏳ 人生进度"）。
  - **中部**: 展示已过去天数（大号加粗字体突出）和剩余天数（辅助文字说明）。
  - **底部**: 使用 `stack` 组件搭建自定义进度条展示（表示度过的比例）以及具体的百分比数值。

## 3. 技术实现
- 新建文件 `life-progress.js`：包含具体的 JS 逻辑计算与 Egern DSL 结构生成。
- 新建文件 `life-progress.yaml`：模块配置文件，暴露环境变量供主配置修改，并定义 `scriptings` 和 `widgets`。

## 4. 验证方式
- 检查 JS 脚本生成的 JSON 格式是否完全符合 Egern 官方 Widget DSL 标准。
- 确保所有的布局属性如 `stack`, `text`, `spacer`, 颜色和进度条计算没有异常和 NaN 值（做容错处理）。
